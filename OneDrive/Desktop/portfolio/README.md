# Ali Haider - Portfolio

A modern, responsive portfolio website built with React, Tailwind CSS, and Lucide icons.

## Features

- **Responsive Design**: Works seamlessly across all devices
- **Smooth Scrolling Navigation**: Single-page application with smooth scrolling between sections
- **Modern UI**: Dark theme with purple/pink gradient accents
- **Interactive Elements**: Hover effects and animations
- **Project Showcase**: Display of featured projects with GitHub links
- **Skills Section**: Categorized technical skills display
- **Contact Information**: Direct links for phone, email, and location

## Technologies Used

- React 18
- Tailwind CSS
- Lucide Icons
- Babel (for JSX transformation)
- HTML5 & CSS3

## Sections

1. **Home**: Hero section with introduction and contact information
2. **About**: Personal description and career goals
3. **Education**: Academic background and qualifications
4. **Experience**: Project experience and achievements
5. **Projects**: Featured development projects
6. **Skills**: Technical and soft skills
7. **Contact**: Contact information and social links

## Deployment

This portfolio is deployed on Netlify. The project uses CDN-hosted dependencies for optimal loading performance.

## Local Development

To run locally:
1. Open `index.html` in a modern web browser
2. All dependencies are loaded from CDNs, so no build process is required
